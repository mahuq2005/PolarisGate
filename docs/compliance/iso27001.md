# ISO 27001 Compliance — PolarisGate

## Overview

PolarisGate aligns with ISO/IEC 27001:2022 Information Security Management System (ISMS) requirements. This document maps PolarisGate controls to ISO 27001 Annex A controls.

## ISMS Scope

The PolarisGate AI Governance Platform, including:
- API Gateway and authentication services
- Guardrails (toxicity, PII, content moderation)
- Hallucination detection and closed-loop learning
- AIDA (Bill C-27) compliance bridge
- Agent governance (sidecar proxy, kill switch)
- Budget control and semantic caching
- ML model management and retraining pipeline
